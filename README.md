# Stash2Jira

A command-line interface to get your repo data from Stash into Jira into a CSV


# Installation

If you don't use `pipsi`, you're missing out.
Here are [installation instructions](https://github.com/mitsuhiko/pipsi#readme).

Simply run:

    $ pipsi install .


# Usage

To use it:

    $ stash2jira --help
